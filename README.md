# Simulador de Elevador

Projeto desenvolvido para a disciplina de **Programação Orientada a Objetos II (POO2)**.

O sistema simula o funcionamento de um elevador em um prédio, controlando chamadas, movimentação entre andares e transporte de pessoas.

## Objetivo

Demonstrar conceitos de **Programação Orientada a Objetos**, como:

* Classes e objetos
* Encapsulamento
* Composição
* Interação entre objetos
* Modularização do código

## Funcionamento

A simulação representa:

* Pessoas em diferentes andares
* Chamadas de elevador
* Movimentação do elevador entre andares
* Entrada e saída de passageiros
* Controle de destinos

Exemplo de simulação:

Pessoa 1 chamou o elevador no andar 2
Pessoa 2 chamou o elevador no andar 1

Elevador no andar 1
Pessoa 2 entrou no elevador

Elevador no andar 5
Pessoa 2 saiu do elevador
