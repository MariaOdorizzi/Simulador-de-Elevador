# Simulador de Elevador

Projeto desenvolvido para a disciplina de Programação Orientada a Objetos II (POO2).

---

## Objetivo

Simular o funcionamento de um elevador em um prédio, aplicando conceitos de:

- Classes e objetos  
- Encapsulamento  
- Composição  
- Interação entre objetos  
- Modularização  

---

## Funcionamento

O sistema simula:

- Pessoas em diferentes andares  
- Chamadas de elevador  
- Movimentação entre andares  
- Entrada e saída de passageiros  

---

## Exemplo de execução

```txt
Simulação iniciada!
Pessoa 1 chamou o elevador no andar 0
Elevador subiu para o andar 1
Elevador subiu para o andar 2
...
Pessoa 1 saiu no andar 5
Simulação finalizada!
